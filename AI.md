# AI Usage

This project uses AI tools as development assistance.

## Tools used
- GitHub Copilot: code completion and boilerplate generation
- ChatGPT: design discussions, debugging help, explanations

## Usage policy
- All code changes are reviewed and understood before being committed
- AI-generated code may be modified or rewritten
- AI is used as a tool, not as an autonomous author

## Scope
AI is used for:
- code suggestions
- refactoring ideas
- debugging assistance
- documentation drafting (README, comments, commit messages)
